/_NumericalFunctions.py_/

Includes numerical calculus functions including Gradient, 2D + 3D Divergence, and 2D + 3D Laplacian.
Also includes Horizontal and Vertical Averaging functions and data slicing functions.


/_PlottingFunctions.py_/

Includes Horizontal and Vertical Contour Plots which ticks from data. Currently, data must be labeled as "data" and must be in xarray format with faces marked with "f" (e.g. "zf") and cell centers marked with "h" (e.g. "zh").
Also, includes functions to make GIF animations and convert GIF to MP4.

/_TestRun.ipynb_/ 

Shows how to import and run functions from above.
